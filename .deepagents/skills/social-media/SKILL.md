# Social Media Post Skill

## Objective
Create shareable, attention-grabbing social media content that aligns with the brand voice and drives engagement.

## Post Types
- **Single posts** (tweet, LinkedIn post, Instagram caption)
- **Threads/series** (multi-part Twitter threads, LinkedIn post series)
- **Callouts** (quick tips, quotes, highlights)

## Inputs to Request
When asking the agent to create social media content, include:
- Topic or theme (e.g., "AI agents for productivity")
- Target platform (Twitter, LinkedIn, Instagram)
- Goal (educate, promote, engage, announce)
- Desired tone (helpful, witty, authoritative, casual)

## Output Structure
1. **Main text**: Clear, concise, and tailored to the platform’s style.
2. **Hashtags**: 3–5 relevant hashtags.
3. **Call to action**: Encourage engagement (comment, share, follow).

### Example Prompt
"Write a LinkedIn post about how AI agents can automate developer documentation. Keep it professional and include a call to action to follow for more AI tips."

## Good Practices
- Keep tweets < 280 characters
- For LinkedIn, use short paragraphs and emojis for scannability
- For threads, start with a strong opening hook and keep each point focused
- Use plain language and avoid jargon unless the audience is technical
